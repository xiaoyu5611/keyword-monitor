🚨 紧急修复：APP划掉后不工作的问题
==========================================

你的问题：
✅ APP打开时输入关键词 → 有警告
❌ APP从后台划掉后 → 没有警告

这说明辅助功能服务被系统杀掉了！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 第一步：确认你用的是最新版APK
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

必须使用：关键词监控-强制引导版.apk

下载地址：
http://38.49.29.167:3000/关键词监控-强制引导版.apk

如果你用的是旧版，请：
1. 卸载当前APP
2. 下载最新版
3. 重新安装
4. 按引导完成设置

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔍 第二步：检查通知栏
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

下拉通知栏，查看是否有：

✅ 应该显示："关键词监控运行中"（常驻通知）

❌ 如果没有显示 → 前台服务没启动
   → 辅助功能需要重新开启

重新启用辅助功能：
1. 设置 → 辅助功能 → 关键词监控
2. 关闭开关
3. 等待3秒
4. 重新打开
5. 返回主屏幕
6. 检查通知栏 → 应该显示"关键词监控运行中"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚡ 第三步：ASUS手机特殊设置（关键！）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ASUS手机系统会自动杀后台，必须做以下设置：

### 1️⃣ 自动启动管理器 ⭐⭐⭐
```
设置 → 电池 → 自动启动管理器
→ 找到"关键词监控"
→ 打开开关（允许自动启动）
```

### 2️⃣ 电池优化 ⭐⭐⭐ 最重要！
```
设置 → 电池 → 电池优化
→ 点击顶部下拉菜单 → 选择"所有应用"
→ 找到"关键词监控"
→ 选择"不优化"
```

或者：
```
设置 → 应用程序 → 关键词监控 → 电池
→ 移除限制
→ 选择"不受限制"
```

### 3️⃣ 移动管理员（手机管家）
```
打开"手机管家"或"移动管理员"APP
→ 省电模式 → 应用程序自动启动
→ 找到"关键词监控"
→ 打开开关
```

### 4️⃣ 锁定后台 ⭐
```
1. 打开最近任务（多任务按钮）
2. 找到"关键词监控"
3. 下拉APP卡片
4. 应该会显示小锁图标（锁定）
```

### 5️⃣ 数据使用权限
```
设置 → 应用程序 → 关键词监控
→ 数据使用情况
→ 允许后台数据使用
```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🧪 第四步：完整测试
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

完成上述设置后：

### 测试1：检查通知栏
□ 下拉通知栏
□ 应该显示"关键词监控运行中"
□ 这个通知不要关闭！

### 测试2：打开APP测试
□ 打开APP，确认状态显示"运行中"
□ 打开微信
□ 输入关键词
□ 刷新管理后台 → 应该有警告 ✅

### 测试3：划掉后台测试（关键！）
□ 打开多任务（最近任务）
□ 确认"关键词监控"已锁定（有小锁图标）
□ 向上划掉"关键词监控"
□ 等待10秒
□ 检查通知栏 → 应该仍然显示"关键词监控运行中" ⭐
□ 如果通知消失了 → 服务被杀了！需要继续设置

□ 打开微信
□ 输入关键词
□ 刷新管理后台 → 应该仍然有警告 ✅

### 测试4：等待测试
□ 划掉APP后，等待5分钟
□ 不要打开任何清理软件
□ 输入关键词测试
□ 应该仍然能触发 ✅

❌ 如果还是不行
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 方案1：不要划掉APP
```
最简单的办法：
• 打开APP后，按Home键
• 不要在多任务中清理APP
• 让APP一直在后台
• 锁定APP（下拉显示锁图标）
```

### 方案2：重启手机
```
完成所有设置后：
1. 重启手机
2. 重启后不要打开APP
3. 检查通知栏 → 应该自动显示"关键词监控运行中"
4. 直接测试输入关键词
5. 应该能触发（说明开机自启成功）
```

### 方案3：使用开发者选项
```
设置 → 关于手机 → 软件信息
→ 连续点击"版本号"7次（开启开发者模式）
→ 返回设置 → 开发者选项
→ 找到"不保留活动"→ 关闭
→ 找到"后台进程限制"→ 选择"标准限制"
```

### 方案4：关闭省电模式
```
确保手机不在省电模式：
设置 → 电池 → 省电模式 → 关闭
```

辅助功能服务工作原理：
1. 启用辅助功能后，服务独立运行
2. 前台服务显示常驻通知
3. 不依赖APP是否打开
4. 但系统可能杀掉服务！

关键标志：
✅ 通知栏显示"关键词监控运行中" = 服务在运行
❌ 通知栏没有显示 = 服务被杀了

如何判断服务是否在运行：
→ 看通知栏！
→ 如果通知在，就一定在运行
→ 如果通知没了，就是被系统杀了
