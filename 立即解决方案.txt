你的手机：ASUS ASUSAI2501A
问题：APP划掉后就不监控了

### 第1步：重要设置（必做！）⭐⭐⭐

#### 设置1：电池优化（最关键！）
```
1. 打开手机"设置"
2. 搜索"电池优化"
3. 点击顶部下拉菜单 → "所有应用"
4. 找到"关键词监控"
5. 选择"不优化"

或者：
设置 → 应用 → 关键词监控 → 电池使用情况
→ 移除限制 → 选择"不受限制"
```

#### 设置2：自动启动
```
设置 → 电池 → 自动启动管理器
→ 找到"关键词监控"
→ 打开开关
```

#### 设置3：锁定后台
```
1. 按多任务按钮（方块键）
2. 找到"关键词监控"APP
3. 下拉APP卡片（向下拉）
4. 会显示一个小锁图标
5. 这样就锁定了，系统不会清理
```

### 第2步：重启辅助功能
```
设置 → 辅助功能 → 关键词监控
→ 关闭开关
→ 等5秒
→ 重新打开
→ 允许所有权限
```

### 第3步：检查通知栏
```
下拉通知栏，应该看到：
"关键词监控运行中"（常驻通知）

⚠️ 这个通知不要关闭！
⚠️ 有这个通知 = 服务在运行
⚠️ 没有这个通知 = 服务被杀了
```

### 第4步：测试
```
1. 打开多任务，划掉APP
2. 等10秒
3. 检查通知栏 → 应该仍然有"关键词监控运行中"
4. 打开微信，输入关键词
5. 刷新管理后台 → 应该有警告
```

❌ 如果还是不行

### 终极方案1：不要划掉APP
```
最简单有效：
1. 打开APP
2. 按Home键（回到桌面）
3. 不要在多任务中清理APP
4. 让它一直在后台
5. 在多任务中锁定它（下拉显示锁）

这样100%有效！
```

### 终极方案2：重启手机
```
完成上述设置后：
1. 重启手机
2. 重启后直接测试（不要打开APP）
3. 输入关键词
4. 检查是否触发

如果重启后能自动工作 → 说明开机自启成功
```

### 终极方案3：关闭省电模式
```
设置 → 电池 → 省电模式 → 关闭

省电模式会杀后台！
```
